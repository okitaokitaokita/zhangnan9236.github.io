Smart City Surveillance Video Analysis System

Intelligent monitoring system that analyzes urban surveillance videos to detect abnormal events (e.g., sudden crowd running, traffic congestion, unauthorized intrusion) and deliver quantitative traffic statistics to improve city safety and management efficiency.

Problem

Urban surveillance systems generate massive amounts of video data that are difficult to monitor manually in real-time. City safety departments need an automated intelligent monitoring system that can detect abnormal events and provide quantitative traffic statistics. This project aims to improve urban safety and traffic management efficiency by replacing manual monitoring with AI-driven analytics.

Data

VIRAT public dataset — a large-scale outdoor surveillance video benchmark containing multiple scenes (parking lots, intersections, public areas) with frame-level annotations for 12+ activity types including pedestrian walking, vehicle turning, and group interactions. Total dataset includes hundreds of video clips at various resolutions and lighting conditions.

Approach

Designed a multi-stage video analytics pipeline: (1) Implemented adaptive background subtraction using multi-period Gaussian Mixture Models (GMM) to handle varying illumination and low-light conditions; (2) Applied optical flow algorithms (TV-L1 and CLG) for dense motion estimation to detect movement patterns; (3) Integrated YOLOv5 for real-time object detection and tracking of pedestrians and vehicles; (4) Developed rule-based event detection logic combining motion magnitude, object density, and trajectory analysis for anomaly recognition; (5) Built a visualization dashboard generating hourly traffic flow curves and group size distribution histograms.

Outcome / Impact

Achieved object detection accuracy of mAP@0.5 > 0.85 on the VIRAT test set; reduced false positive rate by 30% compared to single-model baseline through multi-model fusion. Generated interpretable hourly traffic flow curves and group size histograms validated by simulated city planner feedback as useful decision-support tools. The system processes video at near real-time speed (~15 FPS on GPU).

My Contribution

Led the video preprocessing and background modeling module; implemented the adaptive GMM background subtraction algorithm; developed the optical flow-based motion analysis component; created all traffic statistics visualizations (matplotlib/seaborn); conducted systematic evaluation and validation of the event detection pipeline against ground-truth annotations.

Dec 2022 – Jun 2023 Individual Project (BSc Undergraduate Thesis)

Stock Price Prediction using LSTM

End-to-end deep learning pipeline that uses LSTM networks to forecast short-term stock prices from historical price data and engineered technical indicators, outperforming traditional ARIMA and linear regression baselines on Chinese A-share market data.

Problem

Financial markets exhibit complex non-linear temporal dynamics that traditional statistical models (e.g., ARIMA) struggle to capture. Investors and quantitative analysts need more accurate short-term stock price forecasting tools to support data-driven investment decisions. This thesis investigated whether deep learning approaches, specifically LSTM networks, can outperform traditional time series models for stock price prediction.

Data

Historical daily stock price data from Yahoo Finance API, covering 5+ years (2017–2022) for selected stocks on the Chinese A-share market. Features include open, high, low, close prices, and trading volume — totaling approximately 1,200+ trading day records per stock. Additional technical indicators (MA5, MA20, RSI, MACD) were engineered as supplementary features.

Approach

Built an end-to-end deep learning pipeline: (1) Collected and cleaned raw price data using Python (yfinance, pandas); (2) Engineered technical indicator features (moving averages, RSI, MACD); (3) Applied Min-Max normalization and sliding window technique (window size = 60 days); (4) Designed a multi-layer LSTM architecture (2 LSTM layers + dropout + dense output) using PyTorch; (5) Performed grid search hyperparameter tuning on learning rate, hidden units, and dropout rate; (6) Evaluated with RMSE, MAE, and directional accuracy metrics; (7) Compared performance against ARIMA and simple linear regression baselines.

Outcome / Impact

The LSTM model achieved RMSE 40% lower than ARIMA and 55% lower than linear regression on the test set; directional prediction accuracy reached 62%, providing a meaningful edge over the random baseline (50%). Model demonstrated stability through 5-fold time-series cross-validation with consistent performance across different market conditions. Results were documented in the undergraduate thesis and received an excellent grade.

My Contribution

Independently completed the entire project pipeline from problem formulation through final presentation — including literature review, data collection and cleaning, feature engineering, model architecture design, training and hyperparameter optimization, comprehensive evaluation, results visualization, and thesis writing.
